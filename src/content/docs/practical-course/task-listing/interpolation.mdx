## What is interpolation?

Interpolation is the solution for evaluating a JavaScript expression in the HTML Template.
While some of your HTML content may be static, you might want to display dynamic data in your view.
This allows us to display the value of a property defined in the `component.ts` file in the HTML Template.

Here's your desired situation: display the details of each task!

You'll use the `{{ }}` delimiters to identify the expression to evaluate.
You can use any valid TypeScript expression between these interpolation delimiters.

```html
<div>{{ name }}</div>
=> Gerome
<div>{{ 1 + 1 }}</div>
=> 2
<div>{{ 'Hello ' + name }}</div>
=> Hello Gerome
```
