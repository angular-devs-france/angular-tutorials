## Angular CLI

You already used the CLI to serve the application locally with the `ng serve` command in the <a href="/practical-course/getting-started/#running-the-application" target="_blank">Getting Started</a> step. Now let's use it to create a new component!

The Angular CLI provides a command to generate a new component with the `ng generate component component-name` command.

#### üéì Instructions

1. Run the following command in a terminal to generate a new component called `task-list`

```bash
ng generate component task-list
```

2. Check for a new folder called `task-list` in the `src/app` folder.


:::note
This new folder contains the three following files:

- `task-list.component.ts`
- `task-list.component.html`
- `task-list.component.css`

That's the same structure as the AppComponent component.
:::

## Component metadata

By opening the `task-list.component.ts` file, you can see the following code:

```typescript
// task-list.component.ts
@Component({
  selector: 'app-task-list',
  templateUrl: './task-list.component.html',
  styleUrls: ['./task-list.component.css']
})
export class TaskListComponent implements OnInit {

  ngOnInit() {}
}
```

The `@Component` decorator defines the metadata for the component.

The `selector` property defines the component's CSS element selector.
By using this selector in a template HTML file, Angular creates and displays an instance of the component.

The `templateUrl` property defines the HTML template file for the component.
That's the HTML template that will be displayed when the css selector is used in another HTML file.

The `styleUrls` property defines the CSS files for the component.
These styles will be applied to the component HTML template.


## Display the component

So far the application only displays the `app` component HTML content in the browser.
We generated the new component but we need to display its HTML content by using its selector.

#### üéì Instructions

1. Open the `src/app/app.component.html` file.
2. Add the task list selector as followed:

```html ins={"Add the task list selector (and the main tag for styling and semantic purposes)": 5-8}
// app.component.html
<header>
    <h1>Angular Practical course</h1>
</header>

<main class="container pt-4">
  <app-task-list />
</main>
```

3. Check out the changes in your browser.
